# **App Name**: DJ Glitch

## Core Features:

- Streaming Integration: Connect to major streaming services like SoundCloud, Spotify, Apple Music, TIDAL, and YouTube Music to pull playlists and followed artists.
- AI DJ Mix Assistant: An AI-powered tool to assist with beat-matching, tempo syncing, auto-EQ, compression, and intelligent transition advice.
- Interactive Mixer View: A dual-deck layout with interactive platters, crossfader, EQ, FX racks, loop, hot-cue, and sampler pads.
- Universal Playlist Intelligence: Generate playlists and sets based on mood, event, or context.
- Audio Playback and Mixing Engine: Handles playback functionality using Tone.js for beat-matching and tempo syncing.
- Profile Creation: Allow users to create profiles and share playlists.

## Style Guidelines:

- Primary color: Aquaberry (#02D3E9) for a vibrant cyberpunk feel.
- Background color: Desaturated Aquaberry (#254A4D), a dark backdrop that complements the neon color scheme.
- Accent color: Loki Lime (#06FF00) for interactive elements and highlights, ensuring high contrast.
- Body and headline font: 'Space Grotesk' (sans-serif) for a modern, techy look; pair with 'Inter' for long lines of text if needed.
- Code font: 'Source Code Pro' (monospace) for displaying any code snippets clearly.
- Use neon-style icons that reflect the cyberpunk aesthetic, with animated elements to add interactivity.
- Incorporate smooth transitions and reactive animations to enhance user experience, particularly in the mixer and player views.
